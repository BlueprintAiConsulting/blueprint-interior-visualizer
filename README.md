# BlueprintEnvision Interior Remodel Visualizer

AI-powered interior remodel visualizer for kitchens, bathrooms, living rooms, and bedrooms.  
Multi-pass rendering pipeline: Paint → Cabinets → Countertops → Flooring.  
Presented by MBP Contracting.

## Stack
- **Frontend:** React 19 + Vite + TypeScript
- **Backend:** Express API proxy → Gemini
- **Deploy:** GitHub Pages (frontend) + Render (API)

## Live
- Frontend: https://blueprintaiconsulting.github.io/blueprint-interior-visualizer/
- API: https://blueprint-interior-visualizer.onrender.com/api/ping
